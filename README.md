# 🌌 Deepak Kumar | Professional Portfolio

> A modern, visually sophisticated, and highly interactive personal portfolio showcasing my skills, projects, and professional journey.

## ✨ Key Features

- **Immersive Visuals:** Features a custom 'Nebula' and 'Particle' background animation powered by `tsparticles` and custom canvas elements for a unique, premium feel.
- **Fluid Animations:** Utilizes `framer-motion` for smooth page transitions, stagger effects, and scroll-activated animations.
- **Custom Cursor:** Enhanced user experience with a custom, custom reactive cursor for a sophisticated touch.
- **Responsive Design:** Fully responsive layout built with Tailwind CSS, ensuring a perfect viewing experience across all devices (mobile to 4K displays).
- **Component-Based Architecture:** Scalable and clean codebase utilizing React functional components and hooks.

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [React Type Animation](https://github.com/maximillian-walton/react-type-animation)
- **Visual Effects:** [tsParticles](https://particles.js.org/) & [Three.js](https://threejs.org/)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)

## 📂 Project Structure

```text
frontend/
├── src/
│   ├── assets/       # Static assets like images and videos
│   ├── components/   # Reusable UI components (Navbar, Footer, CustomCursor, Backgrounds)
│   ├── pages/        # Main application pages:
│   │   ├── Home.jsx         # Landing page with typing effects
│   │   ├── About.jsx        # Personal background and video introduction
│   │   ├── Skills.jsx       # Technical skills grid
│   │   ├── Projects.jsx     # Featured projects showcase
│   │   ├── Education.jsx    # Academic history
│   │   ├── Certificates.jsx # Professional certifications
│   │   └── Contact.jsx      # Contact form and social links
│   ├── App.jsx       # Main application layout and routing
│   └── main.jsx      # React DOM entry point
```
